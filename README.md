# Simple Calculator
This is a basic calculator built using Windows Forms in C#. It Performs simple arithmetic operations .

## Purpose
to practice using C# and Windows Forms as part of my learning journey

## Technologies Used
- **Programming Language:** C#
- **Framework:** .NET
- **IDE:** Visual Studio
- **UI Technology:** Windows Forms

## Features

### Binary Operations
- Addition
- Subtraction
- Multiplication
- Divition

### Unary Operations
- Square
- Square root
- Reciprocal

### Other
- Delete, Clear and ClearEntery buttons
- Flip Sign button
- Percent button
- '.' sign button

## How to Run
Clone the repository
